# AI X Automation

このプロジェクトは、自律的に動作する複数のAIエージェント（Researcher, Planner, Writer, Editorなど）を協調させ、日々のX投稿の作成と自動投稿パイプラインを構築するテンプレートです。

## 全体アーキテクチャ

このツールでは、CLIエージェント（Claude Code等）を用いて複数のエージェントを連続して呼び出し、役割分担させながらコンテンツを生成します。

*   **Researcher（リサーチャー）**: 指定されたキーワードやペルソナに基づきWebからトレンド調査や競合アカウントの傾向を分析します。
*   **Planner（プランナー）**: リサーチ結果を踏まえ、ユーザーの反応が取れそうな投稿プランを作成します。
*   **Writer（ライター）**: プランとスタイルガイドに基づいてX投稿のドラフトを作成します。
*   **Editor（エディター）**: ドラフトの品質チェックを行い、承認されれば `approved_post.json` に保存します。
*   (必要に応じて) **Analyst（アナリスト）** や **Community Manager（コミュニティマネージャー）** などがデータから戦略を更新します。

それぞれの生成データは `data/` フォルダ内のJSONを通して次のエージェントへと受け渡されます。

## 前提条件

*   bash 環境 (Linux, macOS, Windows WSL 等)
*   [Claude Code](https://github.com/anthropics/claude-code) (CLIツール) がインストールされ、APIキー設定済みであること
*   `jq`, `curl` コマンドが利用可能であること
*   自動投稿用の[n8n](https://n8n.io/) Webhook等の受信サーバーが存在すること

## 初期セットアップ手順

新規で導入する場合は、以下のセットアップ用スクリプトを1回実行するだけで、AI（Claude Code）が自動的にアカウントを分析し、環境変数やペルソナなどの全初期設定を完了させます。

**実行コマンド**:
```bash
# 引数として "n8n Webhook URL" と "運用するXアカウントID" を渡してください
bash scripts/00_run_setup.sh "https://your-webhook-url-here..." "@your_x_account"
```
※ SSH等の外部環境からの自動構築にも対応しています。（引数が不足している場合はエラーになります）

**自動セットアップ完了後の微調整 (オプション)**

自動化された内容については、後から手動で自由に微調整が可能です。
*   `data/persona.md`: AIが推測したあなたの強みや競合アカウント情報などの戦略が書き込まれているので内容を確認し、必要に応じて加筆・修正してください。
*   `data/style_guide.md`: トーン＆マナーやよく使う語尾などの文体設定を確認します。
*   `data/strategy.md`: 投稿時間やアカウント運用の方針を設定します。

**ログと履歴のクリーンアップ**

テスト稼働時の過去データ等が残っている場合は、必要に応じて `data/` 配下にある以下のファイルを空配列 `[]` にリセットしてください。
* `analytics.json`, `trends.json`, `research_history.json`, `approved_post.json` 等

## 実行方法

セットアップ完了後、一連のパイプラインを実行して投稿を自動生成するには以下のスクリプトを利用します。

```bash
cd scripts/
./pipeline_morning.sh
```

このスクリプトは、1〜5のステップ（リサーチ〜プラン〜ライティング〜エディター承認〜Webhook送信）を順番に実行します。n8nのCronノード等から毎朝定期的に実行することで完全自動化が可能です。
